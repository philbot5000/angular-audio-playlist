angular-audio-playlist

  Easy to Use

angular-audio-playlist is easy to implement and use

    
      <audio playlist="myRadPlaylist"></audio>

  Swap Playlists From Anywhere

Inject audioPlaylistService to any controller and set a playlist or single track to a master audio player.

    
      angular.module('myApp')
        .controller('SomeCtrl', function ($scope, ngAudioPlayerService) {

          $scope.myRadPlaylist = function() {

            audioPlaylistService.setPlaylist([
              {
                order: 0,
                name: 'David Bowie',
                song: 'John, I\'m only dancing',
                src: 'audio/bowie.mp3',
                url: 'http://url.to/some.mp3',
              },
              {
                order: 1,
                name: 'The Gap Band',
                song: 'You Dropped a Bomb on me',
                src: 'http://url.to/some.mp3'
              },
              {
                order: 2,
                name: 'Roy Orbison',
                song: 'Only the Lonely',
                src: 'http://url.to/some.mp3'
              }
            ], 2);
          };
        });

  Themeable

Customize your own template and implement only the features you care about. .

    
      <div>
        <div>
          <button type="button">
            <span ng-bind-html="player.buttonIcon"></span>
          </button>
        </div>
        <div>{{currentlyPlaying.name}}</div>
        <div>{{currentlyPlaying.song}}</div>
        <input type="range" name="slider" min="0" ng-model="percentElapsed" max="100">
        <span>Loaded: {{percentLoaded+'%'}}</span>
        <span>Elapsed: {{percentElapsed+'%'}}</span>
        <span>{{currentTime}}</span>
        <span>{{remainingTime}}</span>
      </div>

  Simple to Extend

At the very least your playlist items must contain a src: '' attribute, but the rest is up to you. Add the information you want and refer to it in your template.
